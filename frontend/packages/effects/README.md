## Effects 目录

`effects` 目录专门用于存放与轻微耦合相关的程式碼和逻辑。如果你的包具有以下特点，建议将其放置在 `effects` 目录下：

- **状态管理**：使用状态管理框架 `pinia`，并包含处理副作用（如非同步操作、API 调用）的部分。
- **使用者偏好设置**：使用 `@vben-core/preferences` 处理使用者偏好设置，涉及本地存储或浏览器缓存逻辑（如使用 `localStorage`）。
- **导航和路由**：处理导航、頁面跳轉等场景，需要管理路由变化的逻辑。
- **元件库依賴**：包含与特定元件库紧密耦合或依賴大型倉庫的部分。

通过将相关程式碼归类到 `effects` 目录，可以使專案结构更加清晰，便于维护和扩展。
